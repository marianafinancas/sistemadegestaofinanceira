🔥 Configurar Firebase

Cole as credenciais do seu projeto Firebase para ativar o armazenamento na nuvem. Faça isso uma única vez.

Como obter:
1. Acesse console.firebase.google.com
2. Crie um projeto → Configurações → Adicionar app Web
3. Copie o objeto firebaseConfig e cole os campos abaixo
4. No menu lateral: Firestore Database → Criar banco de dados (modo produção)
5. Authentication → Provedores → E-mail/senha → Ativar
